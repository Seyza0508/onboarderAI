# Troubleshooting FAQ

## Q: I cannot clone private repositories
Check GitHub org invite and team membership first. Confirm your SSH key is uploaded.

## Q: VPN connects but internal wiki does not load
Disconnect and reconnect VPN, then flush DNS. Verify system time is correct.

## Q: Docker containers fail on startup
Ensure Docker engine is running and no port conflicts exist.

## Q: Migrations fail with auth error
Check local database credentials in environment file and rotate stale secrets.

## Q: I completed setup tasks but do not know my next task
Ask your mentor for a `good-first-task` or review open onboarding-labeled issues.

## Q: Who owns access issues?
Developer Productivity owns GitHub access. IT Network Ops owns VPN.
