# GitHub Access Guide

## Purpose
This guide explains how new hires get access to Northstar GitHub repositories.

## Prerequisites
- Active SSO account
- Completed security training
- Manager assignment in HRIS

## Steps
1. Accept GitHub organization invite sent to company email.
2. Sign in with SSO.
3. Request team membership through Access Portal.
4. Confirm access to required repos:
   - `payments-api`
   - `payments-web`
   - `dev-platform-config`

## Expected Timeline
- Organization invite: within 2 hours of account creation.
- Team membership: 4 to 24 hours.

## Common Issues
- Invite expired: ask Developer Productivity to resend.
- No team visibility: confirm manager submitted role mapping.
- SSO loop: clear browser session and re-authenticate.

## Escalation Path
- Primary: Developer Productivity
- Secondary: IT Identity Team

## Contact
Use `#developer-productivity` and mention your manager and start date.
